## Supported Conversions:

- Base64 Encode & Decode
- ROT13 Encode & Decode
- URL Encode & Decode
- ASCII ⇐⇒ Binary
- ASCII ⇐⇒ Hex
- Binary ⇐⇒ Decimal
- Binary ⇐⇒ Hex
- Decimal ⇐⇒ Fraction
- Decimal ⇐⇒ Percent
- Decimal ⇐⇒ Octal
- Decimal ⇐⇒ Hex
- Fraction ⇐⇒ Percent
- Percent ⇐⇒ PPM
- PPM ⇐⇒ PPB
- PPM ⇐⇒ PPT
- Date ⇐⇒ Roman
- Number ⇐⇒ Roman
